# **App Name**: Moto-Verify

## Core Features:

- Task Creation: Allow admins to create new Moto tasks with detailed descriptions and associated metadata.
- Task Editing: Enable admins to modify existing tasks, including their descriptions, status, and assignments.
- Task Management: Provide an interface to manage tasks, mark them as complete, and track their progress.
- Firebase Integration: Use Firebase Firestore to store task data, ensuring real-time updates and data persistence.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a professional and trustworthy feel.
- Background color: Light gray (#ECEFF1) for a clean and modern look.
- Accent color: Teal (#009688) to highlight key actions and elements, providing visual interest.
- Body and headline font: 'Inter', a sans-serif font providing a clean and modern aesthetic.
- Use consistent, simple icons from a library like Material Icons for a unified visual language.
- Implement a responsive layout with a sidebar navigation for key functionalities.
- Incorporate subtle transitions and animations to enhance user interaction, such as loading spinners and feedback animations on form submissions.
